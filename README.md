# Security Awareness Brochure Project

## Overview
This is a **Python CLI project** designed to generate a professional **cybersecurity awareness brochure** aimed at non-technical users. It compiles essential security best practices into a clean, structured, and easy-to-read Markdown guide.

## Features
* **Modular Content**: The educational content is separated into a `content.py` file, making it incredibly easy to update, add new topics, or translate the brochure without touching the generation logic.
* **Markdown Generation**: The Python script automatically formats the raw text into a beautifully structured Markdown file (`Cybersecurity_Awareness_Guide.md`), complete with a Table of Contents, emojis, and bolded tips.
* **Comprehensive Topics**: Covers critical everyday security topics including Phishing, MFA, Public Wi-Fi, Password Managers, and Social Engineering.

## Requirements
* Python 3.x
* No external dependencies (uses standard library only)

## Usage
1. Open a terminal/command prompt.
2. Navigate to the project directory.
3. Run the application:
   ```bash
   python app.py
   ```
4. Confirm the prompt to generate the brochure. 
5. Open the resulting `Cybersecurity_Awareness_Guide.md` file in any Markdown viewer (or convert it to a PDF for physical distribution).

## Disclaimer
This project is for educational and defensive purposes only. The information provided in the brochure is intended as general guidance for personal cybersecurity hygiene.
